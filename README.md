## 1. **Как работают фриспины?** 🎡

### 1.1 **Типы фриспинов** 🔄

Существует несколько типов фриспинов, которые могут быть предложены игрокам:
- **Фриспины без депозита** — бесплатные вращения, которые можно получить без внесения депозита. Это отличный способ начать играть, не рискуя собственными деньгами.
- **Фриспины при депозите** — бонус, который активируется после внесения депозита. Казино обычно предлагает такой бонус как часть приветственного пакета или во время акций.
- **Фриспины в рамках акции** — фриспины, которые даются в рамках специальных акций, например, в честь праздников или других событий.

### 1.2 **Как использовать фриспины?** 🎰

Для использования фриспинов необходимо:
1. Получить бонус фриспинов через регистрацию, депозит или участие в акции.
2. Перейти в слот, на который распространяются фриспины (обычно информация о слоте указывается в условиях бонуса).
3. Запустить вращение барабанов — фриспины автоматически активируются и используют на каждом вращении.

При выигрыше с фриспинов, вы можете вывести эти деньги, если выполнены условия по отыгрышу бонуса.
